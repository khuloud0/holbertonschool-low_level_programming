# structures_typedef

## Task 0: Define a new type `struct dog`

This project defines a structure called `struct dog` with the following members:
- `char *name` : dog's name
- `float age` : dog's age
- `char *owner` : dog's owner

### File:
- `dog.h`

### Example:
```c
struct dog my_dog;

my_dog.name = "Poppy";
my_dog.age = 3.5;
my_dog.owner = "Bob";
